# Docs

Guides and reference material for the Last Oasis Modkit. Files at the top level here are repo-original; [`modkit-guides/`](modkit-guides/) holds expanded community variants of the official Donkey Crew Drive docs.

## At a glance

| Doc | What it's for |
| --- | --- |
| [`modkit-python-scripting.md`](modkit-python-scripting.md) | Enabling the Python plugin in the Modkit, three ways to run a script, what each script in [`../scripts/modkit/`](../scripts/modkit/) does, UE 4.25 reflection gotchas, "your own extractor" template. |
| [`myrealm-configuration.md`](myrealm-configuration.md) | Field-by-field reference for the [MyRealm portal](https://myrealm.lastoasis.gg/): identity, access, hosting mode, events, gameplay tuning (multipliers, claims, combat, decay, economy, PvP), the `Mods=` field, end-to-end checklist for a new modded realm. |
| [`modkit-guides/`](modkit-guides/) | Five expanded variants of the official Donkey Crew guides (mod authoring, hosting, custom maps, references, v2→v3 porting). [Index](modkit-guides/README.md). |

Top-level repo overview: [`../README.md`](../README.md).
