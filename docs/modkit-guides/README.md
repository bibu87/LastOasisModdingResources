# Modkit Guides

Improved, expanded variants of the official Donkey Crew Modkit docs in the [Last Oasis Modkit Drive folder](https://drive.google.com/drive/folders/1QqS5Z32g07FLpTja2g6oCUJ3YeJnqND1). Each guide preserves the original content and adds field-by-field references, command-line breakdowns, troubleshooting sections, and cross-links between related topics.
